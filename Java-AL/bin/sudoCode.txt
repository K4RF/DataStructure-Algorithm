Path(LCS 저장하기 리스트)

for(i -> 1 ~ A 문자열 길이){
    for(j -> 1 ~ B 문자열 길이){
        A[i]와 B[i]가 같으면 DP[i][j]의 값을 왼쪽 대각선 값 + 1로 저장하기
        다른 경우에는 왼쪽의 값과 위의 값 중 큰 값으로 DP[i][j] 채우기
    }
}
