D[N][L][R](빌딩 N개를 왼쪽에서 L개, 오른쪽에서 R개가 보이도록 배치할 수 있는 모든 경우의 수)
D[1][1][1] = 1

for(i -> 2 ~ N){
    for(j -> 1 ~ L){
        for(k -> 1 ~ R){
            D[i][j][k] = D[i - 1][j][k] * (i - 2) + D[i-1][j][k-1] + D[i-1][j-1][k]
            나온 결과를 1000000007 나머지 연산 수행하기
        }
    }
}
