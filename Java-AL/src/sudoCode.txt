int excute(s, e){
    result(결괏값)
    if(이미 계산한 구간일 때) DP[i][j] 값 바로 리턴
    if(1개 행렬일 때) 연산 횟수 0 리턴
    if(행렬이 2개일 때) 2개 행렬 연산값 리턴
    if(행렬이 3개 이상일 때){
        for(i -> s ~ e){
            excute(s, i) + excute(i + 1, e) + 앞뒤 구간의 행렬을 합치기 위한 연산 횟수
        }
    }
}
